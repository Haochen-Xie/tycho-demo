EclipseCon 2011 Tycho Tutorial

Setup/Prerequisites

  1. You need an Indigo SR1 eclipse classic SDK + m2eclipse + m2e Tycho configurators installed.
    For your convenience, we provide pre-packaged zips/tarballs with this installation in the root folder of the USB stick.
  2. Copy and unpack the eclipse SDK archive for your platform and start it.
  3. Copy folders exercises/ and repository/ to your local harddrive

Exercises Overview: Building an RCP application in 5 step-by-step exercises

The hands-on exercises will guide you through setting up a typical tycho project for an RCP application.
  • We start from scratch with a small "Hello World" RCP plugin,
  • then we add a simple unit test to make sure the plugin works
  • then we add a feature for grouping the plugins we want to deliver.
  • Next, we create a p2 repository/update site containing our feature(s) and bundles,
  • and finally we add a product with native launcher binaries, branding etc. for which we create pre-packaged zip installations.

Table Of Contents

The exercises consist of 5 steps:
  1. Build a simple RCP plugin
  2. Build and execute a simple test
  3. Add a feature
  4. Add a repository
  5. Add a product (solution)

Each step has a subfolder with stub content as a starting point.
The links above give detailed step-by-step instructions for each step.
The folder for an exercise is at the same time starting point for the current exercise and solution of the previous one.
If you want to catch up to a certain step during the exercises, just follow the recovery instructions provided in the corresponding step.

If you successfully installed everything under "Setup" above, you can now jump to the first exercise.
